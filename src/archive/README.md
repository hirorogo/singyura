- **main_improved.py** - Phase 1/2改善版AI
  - **性能**: 31-39%の勝率
  - **問題**: 過剰な最適化により性能が低下
  - **アーカイブ理由**: シンプル版（38-45%）に劣る性能

主な問題点：

- パス観測の制約が厳しすぎる
