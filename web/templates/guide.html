🤖 ボット導入ガイド

七並べAIの導入方法を詳しく説明します

📦 必要な環境

必須

  • Python 3.7以上
  • numpy

オプション（開発用）

  • CuPy (GPU高速化用)
  • PyTorch (GPU高速化用)

🔧 インストール手順

ステップ 1: リポジトリのクローン

git clone https://github.com/hirorogo/singyura.git
cd singyura

ステップ 2: 依存パッケージのインストール

# 必須パッケージのみ
pip install numpy

# GPU開発用（オプション）
pip install cupy-cuda12x  # NVIDIA GPU
# または
pip install torch  # NVIDIA/Apple Silicon

ステップ 3: 動作確認

# スクリプト版を実行
python src/submission.py

# ベンチマーク実行
cd src
python benchmark.py

📓 Jupyter Notebook での使用

ローカル環境

# Jupyter Notebookで開く
jupyter notebook 提出用/teishutu.ipynb

Google Colab での使用

  1. Google Colab にアクセス
  2. GitHubから直接開く：File > Open notebook > GitHub
  3. hirorogo/singyura で検索
  4. 提出用/teishutu.ipynb を選択

⚙️ 設定のカスタマイズ

シミュレーション回数の調整

# submission.py または teishutu.ipynb 内で
SIMULATION_COUNT = 300  # 推奨値

# パフォーマンスガイド:
# 100:  高速（勝率50-55%）
# 300:  推奨（勝率55-60%）
# 700:  最強（勝率80%）※処理時間増

Phase 1改善フラグ

ENABLE_PASS_REMOVAL = True              # PASS除外
ENABLE_WEIGHTED_DETERMINIZATION = True  # 重み付け確定化
ENABLE_ADAPTIVE_ROLLOUT = True          # 適応的ロールアウト

🎯 使い方

1. AIの実装

my_AI 関数が既に実装されています。この関数が各ターンで呼び出されます。

2. テスト実行

# ベンチマークでAIをテスト
cd src
python benchmark.py

3. 大会への提出

Google Colab で 提出用/teishutu.ipynb を実行し、my_AIセルのコードをコピーして提出してください。

❓ トラブルシューティング

エラー: ModuleNotFoundError

pip install numpy

メモリ不足

# SIMULATION_COUNT を減らす
SIMULATION_COUNT = 100

動作が遅い

# シミュレーション回数を減らす
SIMULATION_COUNT = 100
